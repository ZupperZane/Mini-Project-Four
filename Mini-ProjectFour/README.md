#### This website allows users to filter a database list of movies based on:
* Title (default)
* Director
* Genre
* Release year
* IMDB Rating
* Runtime
* Age group
* Language
* Budget
* Description


#### Users are able to search within their chosen category to eliminate movies to find what they are looking for easily
#### By default, movies are sorted and shown to the user A-Z or by ascending number
#### Users also have the ability to add items to their wishlist
#### When there are items in the wishlist, users can download their wishlisted movies in a simple readable .txt file
